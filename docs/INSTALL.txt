wcc.churches Installation
-------------------------

To install wcc.churches using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.churches`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.churches
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
